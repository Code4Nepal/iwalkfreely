# iwalkfreely

### How do I get started with iwalkfreely-webapp?

```
/* global dependencies */
$ npm install -g gulp bower
```

```
$ git clone git@github.com:Code4Nepal/iwalkfreely.git
$ cd iwalkfreely/app
$ npm install
$ bower install
```

### How do I run the app?

```
$ cd app
$ npm start

open http://localhost:8079
```
